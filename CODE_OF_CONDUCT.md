# 社区行为准则

## 我们的承诺

Nora 社区致力于为所有善意参与者提供安全、尊重、公平且专业的协作环境。参与者应尊重不同背景、经验、观点和沟通方式，并以项目和社区的长期健康为共同目标。

## 鼓励的行为

- 友善、诚实并聚焦问题本身
- 清晰表达不同意见，同时尊重他人的经验
- 提供和接受建设性反馈
- 对自己的决定、代码和影响负责
- 发现伤害时主动停止、道歉并修复
- 保护他人的隐私和安全

## 禁止的行为

- 骚扰、威胁、歧视或持续越过他人明确表达的边界
- 人身攻击、侮辱、贬损、恶意挑衅或不必要的政治攻击
- 未经允许公开他人的身份、联系方式、凭据或其他私人信息
- 性化语言、图像或不受欢迎的关注
- 滥用项目渠道、冒充项目成员或报复报告者
- 其他在专业协作环境中明显不适当的行为

## 适用范围

本准则适用于仓库、Issue、Pull Request、评审、讨论区、项目活动，以及个人公开代表 Nora 社区时。

## 报告与执行

不要在公开 Issue 中披露敏感事件细节。请通过仓库托管平台提供的私密联系渠道联系维护者。仓库公开接受贡献前，所有者必须配置并验证一条私密报告渠道；在此之前，本文件仅作为治理占位。

维护者会优先保护报告者隐私，基于事实调查，并按影响采取澄清、警告、限制参与或永久移除等措施。故意虚假报告、报复或泄露调查信息同样违反本准则。

## 来源

本准则以 [Contributor Covenant 3.0](https://www.contributor-covenant.org/version/3/0/) 的价值观和执行原则为基线，并针对 Nora 当前的占位阶段进行了精简。具体执行流程应在社区正式开放前由独立治理 Issue 完成。
